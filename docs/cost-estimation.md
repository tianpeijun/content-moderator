# 商城评论内容审核系统 — AWS 成本估算

> 更新时间：2026-04-18
> 架构变更：纯文本主模型已从 Claude Sonnet 4 切换为 Qwen3 32B，并启用预审过滤（关键词/正则，命中率 60-68%）。

## 业务假设

| 参数 | 值 |
|------|------|
| 日均审核量 | 2,000 条 |
| 月均审核量 | 60,000 条 |
| 纯文本审核占比 | 80%（48,000 条/月） |
| 图文混合审核占比 | 20%（12,000 条/月） |
| 预审命中率（纯文本） | 60%（不调模型，走关键词/正则） |
| 管理后台日活用户 | 3-5 人 |
| 批量测试频率 | 2 次/月，每次 ~500 条 |
| 部署区域 | us-east-1 |

### 流量按模型路径分布

| 路径 | 月调用量 | 占比 | 说明 |
|------|---------|------|------|
| 预审命中（不调模型） | 28,800 | 48% | 60% × 48,000 纯文本 |
| 纯文本 → Qwen3 32B | 19,200 | 32% | 40% × 48,000 纯文本 |
| 图文混合 → Claude Sonnet 4 | 12,000 | 20% | 全部图文 |
| 批量测试 | ~1,000 | — | 另算（预审和模型混合） |

## 一、Amazon Bedrock（AI 模型调用）

> 定价来源：[Amazon Bedrock Pricing](https://aws.amazon.com/bedrock/pricing/)

### 单价

| 模型 | 输入 / 百万 tokens | 输出 / 百万 tokens |
|------|------|------|
| Qwen3 32B | $0.15 | $0.60 |
| Claude Sonnet 4 | $3.00 | $15.00 |
| Claude Haiku 4.5（备用） | $1.00 | $5.00 |

### Token 用量（单次调用估算）

基于真实调用采样：输入约 1,000 tokens（规则 + 标签定义 + 评论内容），输出约 200 tokens（结构化 JSON 响应）。

### 单次调用成本

| 模型 | 单次成本 | 说明 |
|------|---------|------|
| Qwen3 32B | **$0.00027** | 1000 × $0.15/M + 200 × $0.60/M |
| Claude Sonnet 4（实测） | $0.00523 | 基于 201 次真实调用校准 |

### 月度 Bedrock 费用

| 路径 | 月调用量 | 单价 | 月成本 |
|------|---------|------|--------|
| 预审命中 | 28,800 | $0（不调模型） | **$0.00** |
| Qwen3 32B（纯文本非命中） | 19,200 | $0.00027 | $5.18 |
| Sonnet 4（图文） | 12,000 | $0.00523 | $62.76 |
| 批量测试（含预审） | ~1,000 | ≈ $0.002 平均 | $2.00 |
| **Bedrock 合计** | **61,000** | | **$69.94** |

> **相比旧架构节省**：旧架构（Sonnet 4 全量）月度 Bedrock 成本 $320.08。新架构 $69.94，**节省 78%**（$250/月）。主要贡献：预审过滤让 48% 流量零成本，纯文本走 Qwen3 单价降低 95%。

## 二、AWS Lambda（计算）

> 定价：$0.20 / 百万请求，$0.0000166667 / GB-秒。免费额度：每月 100 万请求 + 40 万 GB-秒。

| 函数 | 内存 | 平均耗时 | 月调用 | 月成本 |
|------|------|---------|--------|--------|
| 审核 API | 512 MB | 1.5s（混合） | 60,000 | $0.76 |
| 管理后台 API | 512 MB | 0.5s | 10,000 | $0.04 |
| 批量测试 Worker | 1024 MB | 5s | 1,000 | $0.08 |
| Lambda Authorizer（API Key 校验） | 128 MB | 0.02s | ~12,000（5min 缓存后） | $0.01 |
| **Lambda 合计** | | | | **$0.89** |

> 审核 API 平均耗时从 3s 降到 1.5s（预审命中只 50ms，Qwen3 比 Sonnet 4 快 60%），计算费用随之下降。免费额度内实际费用接近 $0。

| **理论最大** | | $32.63 |
| **实际校准**（4 天实测折算） | | **$5.01** |

## 进一步优化建议

| 优化项 | 预期节省 | 实施成本 |
|--------|---------|----------|
| VPC Endpoint for Bedrock 替代 NAT Gateway | -$0 ~ -$25/月（取决于流量） | CDK 改造 |
